# Telegram 第三方客户端

第三方客户端是指使用 Telegram API 编写的客户端。本文介绍常见的 Telegram 第三方客户端，下载官方客户端请 [点击此处](https://telegram.org/apps) 。

::: tip

由于官方的一些限制，第三方客户端无法发送短信验证码。也就是说，您只能通过已登录设备接收验证码来登录第三方客户端。

:::
